Dabeey's Portfolio Website

This is a Flask-based portfolio website designed to showcase skills, projects, and services. It includes a contact form that allows users to send messages via email.

## Features

- **Home Page**: A welcoming landing page.
- **About Page**: Information about the individual or organization.
- **Resume Page**: A detailed resume or CV section.
- **Services Page**: A list of services offered.
- **Portfolio Page**: A showcase of projects or work samples.
- **Contact Page**: A functional contact form with email integration.

## Technologies Used

- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, JavaScript, Bootstrap
- **Email Integration**: Flask-Mail
- **Environment Configuration**: Python `dotenv` for managing environment variables
